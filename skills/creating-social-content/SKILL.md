# Social Content Creation

Create platform-native social media posts that build audience through authentic storytelling. Each post should pass the test: "Could any other company post this exact thing?" If yes, rewrite.

## When to Use

- A topic, decision, or result needs to become a social post
- Writing posts for a specific platform (X, LinkedIn, TikTok, Instagram)
- Adapting content across platforms

## Process

1. Identify the topic and target platform.
2. Select content pillar if not specified: Founder's Journey (40%), Behind-the-Scenes Ops (25%), Case Studies & Proof (20%), Industry Hot Takes (15%).
3. Write platform-native content:
   - **X single tweet**: Under 280 chars. No external links in main tweet.
   - **X thread**: Numbered (1/, 2/, 3/), 3-7 tweets max. Hook-first opening. End with a question.
   - **LinkedIn**: Professional tone. Line breaks every 1-2 sentences. Hook first line. 150-300 words. 3-5 hashtags.
   - **TikTok script**: Hook in first 3 seconds. Face-to-camera. 45-90 seconds. Visual cues noted. Conversational.
   - **Instagram Reels**: Cross-post from TikTok selectively. Separate caption text.
4. Apply brand voice: direct, specific, honest, aspirational without hype.
5. Verify authenticity -- rewrite anything generic.

## Output Format

- Post content ready to copy-paste, labeled by platform
- Character count (for X)
- Engagement notes (timing, reply strategy)
- Suggested reply or follow-up content (for X threads)

## Constraints

- Never fabricate numbers or results.
- Never use corporate jargon: leverage, synergy, disrupt, paradigm.
- Limit to 1-2 emoji per post maximum; zero is fine.
- Never prefix threads with "Thread:" or "THREAD."
- Always attribute the voice to the founder, not the company generically.
